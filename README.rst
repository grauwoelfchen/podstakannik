Podstakannik
============

Podstakannik (``Подстака́нник``) is a friend for your tea time ;)

This is a file/directory daemon watches personal diary posts on a local
filesystem.


Setup
=====

.. code::zsh

  % cargo install podstakannik


Usage
=====

.. code::zsh

  % podstakannik
